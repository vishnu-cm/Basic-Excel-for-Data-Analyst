#### Excel Sales Analysis Project



**Project Overview**

This project demonstrates practical Excel data analysis skills using a sample sales dataset. The goal is to analyze sales performance, apply business logic using formulas, and summarize insights using Pivot Tables.


Files

* Sales\_Analysis.xlsx – Sales data, formulas, lookups, and pivot tables

Skills Used

Calculated Columns

* Total Sales calculated using:
* Units Sold × Unit Price

Conditional Formulas

* IF formula to classify sales as High or Low
* IF \& AND formulas to identify discount-eligible transactions

Lookup Functions

* VLOOKUP / XLOOKUP used to fetch product warranty details from a reference table

Pivot Table Analysis

* Total sales by region
* Sales performance by sales representative
* Category-wise analysis (Electronics vs Accessories)
* Filters and slicers for interactive analysis

Key Insights

* Identified top-performing regions
* Compared sales representative performance
* Analyzed category contribution to revenue
* Applied business rules using Excel formulas


Tools \& Techniques

* Microsoft Excel
* Pivot Tables \& Filters
* IF, AND formulas
* VLOOKUP / XLOOKUP
* Data cleaning and structuring
